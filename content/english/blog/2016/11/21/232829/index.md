<p>僕は人生で２度、「これで終わりか！？」というほどの、<br/> <br/>大きなピンチを経験したが、もう一生、生活に困ることはないだろう。<br/> </p><p> </p><p>それは、<br/> <br/>・お金があるから幸せになれるわけではない<br/> <br/>・見栄を張るメリットはない<br/> <br/>とうことを嫌というほど、実感できているからだ。<br/> <br/> <br/>僕は馬鹿だから、２０代、３０代でそれなりに成功した時には、<br/> <br/>クルマはもちろん、クルーザーも買ったし、ひと通りの遊びもやった。<br/> <br/>そして、転落。<br/> <br/> <br/>一度ならともかく、二度も同じ失敗をした。<br/> <br/>転落すると、それまで見えなかったことが見えるものだ。<br/> <br/>それは、誰もが想像できることで、<br/> <br/>僕に友人や女性が集まってきているのではなく、「金」に集まってきていたということであり、<br/> <br/>転落した瞬間から彼らの多くは態度を変えた。<br/> <br/> <br/>そうした経験から、お金がない幸せというのも分かるようになった。<br/> <br/>「お金がない幸せ？？」<br/> <br/> <br/>今、世界中で拝金主義がはびこっているが、</p><p> </p><p>それは「お金があれば幸せになれる」という前提に立っている。<br/> <br/>もちろんお金で解決できることが多いのは事実だが、<br/> <br/>それで幸せになれるか、というのは別の話だ。<br/> <br/> <br/>逆に、徹底的にお金がないことの強さというか、幸せというのもある。<br/> <br/> <br/>僕は、最初の起業で裏切りに遭い、自ら起業した会社を追い出され、</p><p> </p><p>２６歳で無一文になった。</p><p> </p><p>その後、しばらくは月収１８万円（手取り１５万円くらい）で政治家の秘書をしたのだが、</p><p> </p><p>つき合っていた彼女の誕生日に、５万円の指輪を贈った。<br/> </p><p><br/>手取り１５万円で一人暮らしをしていた当時の僕にとって、</p><p> </p><p>５万円というのはかなりの出費であり、購入する時にはそれなりに気合が入り、</p><p> </p><p>渡すときにも高揚感があった。</p><p> </p><p>彼女も僕の収入を知っていたから、その痛みを理解してくれていたのか</p><p> </p><p>感動してくれた（と思う。たぶん）。<br/> <br/> <br/>今の僕が、金銭的にそれなりのことをしても、その時のような思いにはなれないだろう。<br/> <br/>相手も高価なものを貰う喜びはあるだろうが、貧しいあの時のような感動はないだろう。<br/> <br/> <br/>カネがないからこその喜びや感動があるのだ。</p><p><br/>カネがないことのメリットは、他にもある。</p><p> </p><p>妬まれることがない。</p><p> </p><p>泥棒や詐欺に遭わない。</p><p> </p><p>借金の依頼が来ない。</p><p> </p><p>裁判で訴えられない。</p><p> </p><p>ケチだと思われない。</p><p> </p><p>他人にお願いしやすい。</p><p> </p><p>服装に気を使わなくていい。</p><p> </p><p>デートやつき合いに金がかからない。</p><p> </p><p>かわいそうだと思ってもらえる。</p><p> </p><p><br/>そう、貧乏というのは、特に大貧乏というのは特権なのだ。</p><p> </p><p> </p><p>「大富豪（大貧民）」というトランプのゲームがある。</p><p> </p><p>ほとんどの人がやったことがあると思うが、簡単にルールを説明すると</p><p> </p><p>配られたカードを弱い順番に出していき、自分のカードがなくなったら勝ち、</p><p> </p><p>というゲームだ。</p><p> </p><p><a href="blog_import_6621b6d5609ba.jpg">{{< figure src="blog_import_6621b6d4303aa.jpg" width="220px" height="147px" >}}</a></p><p><br/>地方によってローカルルールがあるが、カードの強い順に</p><p> </p><p>ジョーカー、２，１，Ｋ、Ｑ、Ｊ、１０、・・・・３、となる。</p><p><br/>このゲームの特徴は、「階級」があることだ。</p><p> </p><p>複数人でゲームをやると、一番勝った者が「大富豪」、次が「富豪」、</p><p> </p><p>あとは「平民」で、一番負けが「大貧民」、次が「貧民」となる。</p><p> </p><p>「大貧民」は「大富豪」に強いカードを２枚献上し、</p><p> </p><p>「大富豪」は「大貧民」に弱いカードを２枚渡す。</p><p> </p><p><br/>これによって、「大富豪」は圧倒的に有利な状態からゲームを進めることができる。</p><p> </p><p>では、「大貧民」は圧倒的に不利なのか、といえばそうともいえない。</p><p> </p><p>それは「大貧民」ならではの特権があるからだ。</p><p> </p><p> </p><p>それは、先行権と並び順。</p><p> </p><p>地方によっても違うのだが、僕の知るルールでは、</p><p> </p><p>大貧民から順に時計回りで並ぶ。</p><p> </p><p>まず、最初に大貧民がカードを出すのだが、</p><p> </p><p>弱いカードばかり集まっていると、戦略によってはそれなりに強い。</p><p> </p><p> </p><p>また、大富豪が隣りにいるから、そこでカードが切れると出しやすい。</p><p> </p><p>下手に中途半端な「貧民」になるくらいなら、「大貧民」のほうが強いのだ。</p><p> </p><p> </p><p>僕は幸か不幸か、成功とドン底を短期間で経験し、貧しさの幸せと強さを知った。</p><p> </p><p>だから中途半端なお金の使い方をしない。</p><p> </p><p>それが無意味であることが分かるからだ。</p><p> </p><p> </p><p>若い頃、調子こいて一緒に遊んでいた仲間の多くは、退場していった。</p><p> </p><p>ブランド物を買い、高級外車に、高級時計、中途半端に金を持つとそういうカネの使い方をする。</p><p> </p><p>その心理は、「金があることで幸せである」という錯覚であり、</p><p> </p><p>「他人より金があることで得る幸福感や優越感、安心感」だ。</p><p> </p><p>そういう人たちは、なんだかんだ言って「不安」なのだ。</p><p> </p><p>そういう奴が危ない。</p><p> </p><p> </p><p>僕は若い頃に、大貧民の幸せ、楽しみ方、を経験して本当に良かったと思う。</p><p> </p><p>しかし、、、</p><p> </p><p>現実の人生にトランプのような「革命」はない。</p><p><br/>「大貧民」「貧民」「平民」「富豪」と１つずつ上がっていくしかないわけだ。</p><p> </p><p> </p><p>ただ、方法によって、効率的に早く上がっていくことはできる。</p><p> </p><p>僕の起業や投資の経験が、そんなことの役に立てれば嬉しい。</p><p><br/>「今からもう一度『大貧民』からやり直せるか？」って？</p><p> </p><p>やれないことはないけど、今さら「大貧民」からは面倒かな。</p><p> </p><p> </p><p><a
