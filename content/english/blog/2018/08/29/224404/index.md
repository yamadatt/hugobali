target="_blank">その１</a>　<a href="https://ameblo.jp/baliclub/entry-12356313120.html" target="_blank">その２</a>　<a href="https://ameblo.jp/baliclub/entry-12356316979.html" target="_blank">その３</a>　<a href="https://ameblo.jp/baliclub/entry-12358172525.html" target="_blank">その６</a>　<a href="https://ameblo.jp/baliclub/entry-12357941478.html" target="_blank">番外編</a></p><p> </p><p> </p><p>昨年末にクビにした４０代の自称コンサル女性スタッフだ。</p><p> </p><p>米国ＭＢＡ取得、米国の不動産会社で上級職のポジションにて勤務、</p><p> </p><p>ホテルフォーシーズンでマネージャー経験あり、</p><p> </p><p>英語、インドネシア語、バリ語、ジャワ語が堪能</p><p> </p><p>インターネット戦略、サイト構築のプロフェショナル</p><p> </p><p>と履歴書は素晴らしく、面接時の立ち振る舞いも完璧だった。</p><p> </p><p><br/>しかし、、働き始めると、成果はゼロ。</p><p> </p><p>さらには、ズル休み、虚偽報告、経歴詐称、横領まで発覚した。</p><p> </p><p>もちろんクビにしたわけだが、その後も酷かった。</p><p> </p><p><a href="blog_import_6621c5df3f6a5.jpg">{{< figure src="blog_import_6621c5df3f6a5.jpg" width="220px" height="313px" >}}</a>　<a href="blog_import_6621c5e08496b.jpg">{{< figure src="blog_import_6621c5e08496b.jpg" width="220px" height="304px" >}}</a></p><p><br/>おとなしく退職してくれればよかったのだが、不当解雇やセクハラなどを</p><p> </p><p>でっち上げ、弁護士を立てて脅してきた。</p><p> </p><p>こちらも弁護士を立て応戦する。</p><p> </p><p>そもそも彼女の言い分はウソだから、相手の弁護士もやりようがない。</p><p> </p><p>事実を知った弁護士が下りたり、思い通りにならないから彼女が解雇したり</p><p> </p><p>を繰り返した。</p><p> </p><p>前回までで、４人の弁護士が登場したが、その後４人を解雇し、</p><p> </p><p>９人目の弁護士が登場した。</p><p> </p><p> </p><p>こちらは弁護士を通して、彼女の不正の証拠を警察に提出し、捜査を依頼。</p><p> </p><p>使途不明金は数十万円あったが、完璧な証拠となると領収書の改ざんの１件で</p><p> </p><p>金額は６万円ほどだった。</p><p> </p><p>金額が小さいから起訴までは難しいと分かっていたが、今後起きうる彼女からの</p><p> </p><p>嫌がらせや誹謗中傷を考えると、放置はできなかった。</p><p> </p><p> </p><p>今回のケースは、金額が小さかったから警察も消極的だったが、そこはインドネシア。</p><p> </p><p>コネと賄賂で捜査してもらうことに成功した。</p><p> </p><p> </p><p>しかし、日本と違って住民登録の制度がないから、彼女の居所がつかめない。</p><p> </p><p>警察に彼女が依頼している弁護士を教え、居所を突き止めた。</p><p> </p><p>と思ったのだが、彼女が弁護士に伝えている住所自体がでたらめだった。</p><p> </p><p> </p><p>警察が弁護士から彼女に連絡させ、捕まえられるかと思ったが</p><p> </p><p>あと一歩のところで逮捕には至らなかった。</p><p> </p><p>彼女が行方をくらましたのだ。</p><p> </p><p> </p><p>その後、弁護士も連絡が取れなくなり、居所が分からなくなった。</p><p> </p><p>たぶん、弁護士が警察が動いてることを彼女に教えたのだろう。</p><p> </p><p>警察が本気になれば指名手配なのだが、被害額が大きくないから</p><p> </p><p>そこまではやらない。</p><p> </p><p>警察は「この女はもう、バリ島にはいないだろう」と言っている。</p><p> </p><p> </p><p>結局、彼女に関わった弁護士は９人。</p><p> </p><p>全員が着手金をもらわず、成功報酬で動いていた。</p><p> </p><p>こちらが外国人だからカネになる、と思ったのだろうが、返り討ちにした。</p><p> </p><p> </p><p>とはいえ、こちらも弁護士を雇い、警察に動かし、カネも手間もかかった。</p><p> </p><p>仕方がなかったとはいえ、バカバカしい。</p><p> </p><p>揉め事にいいことなんてない。</p><p> </p><p>さて、、前向きなことをやるか。</p><p> </p><p><a
