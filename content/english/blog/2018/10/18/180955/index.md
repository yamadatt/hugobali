<p>バリ島現地法人のローカルスタッフが気に入っている飲食店がある。</p><p> </p><p>そこは、サヌール、メイン通りの裏。</p><p> </p><p>夜な夜な異様に賑わっている。</p><p> </p><p>いつも、ローカル若者男子が３００人はいるだろう。</p><p> </p><p>店の名は、"Warung Pan Tantri Sanur"</p><p> </p><p><a href="blog_import_6621c6e90686e.jpg">{{< figure src="blog_import_6621c6e90686e.jpg" width="220px" height="165px" >}}</a> <a href="blog_import_6621c6eaac80f.jpg">{{< figure src="blog_import_6621c6eaac80f.jpg" width="220px" height="165px" >}}</a></p><p><br/>「安くて、いい店だから！」と何度か誘われたことがあるが</p><p> </p><p>日本人の僕としては、話を聞けば聞くほど行く気がしない。</p><p> </p><p>しかし、現地スタッフのタケシは、日本人のくせにこの店が大好きだ。</p><p> </p><p>彼の報告によると、、、。</p><p> </p><p><br/>１、酒はアラック！<br/>　　</p><p>　　ビンタンビール、ジュースなどもあるが、ほぼ全テーブルで頼んでいるのが、</p><p> </p><p>　　ビールボトルに入った地酒、アラックだ。</p><p> </p><p>　　これを、緑色の液体で割って飲む。</p><p> </p><p><br/>　　緑色の正体は、パンダンを煮だして色をつけたもの。</p><p> </p><p>　　パンダンの香りこそするが、基本的には砂糖水、、である。</p><p> </p><p>　　タケシは当然、このセットを頼む。</p><p> </p><p>　　美味くない。　　アラックも、砂糖水も。</p><p>　　</p><p>　　料金は、アラック、ハーフボトルで15,000（120円ほど）。</p><p>　　</p><p>　　安い！！</p><p><br/>　　ちなみに、僕はアラックが好きじゃない。</p><p>　　　</p><p>　　だから訪バリする時は、赤霧島、いいちこ、ジャックダニエルを持ち込んでいる。</p><p> </p><p><br/>２、美味しい料理</p><p> </p><p>　　この店の看板メニューは、アヤムゴレン・テプン（鳥のから揚げ）。</p><p> </p><p>　　鶏のから揚げとはいうものの、中に入っている肉が、異様に少ない。</p><p> </p><p>　　ほとんど衣、というか、揚げた小麦粉。</p><p> </p><p>　　鳥肉入りのかりんとう、といったほうがいい。</p><p> </p><p>　　タケシに言わせると、これが意外に美味いらしく、</p><p> </p><p>　　肉が少ない（中身がない）から満腹にならず、何個も食べられる、とか。</p><p> </p><p>　　奴は病みつきになっているが、僕はイヤだな、そんな食べ物。笑</p><p> </p><p><a href="blog_import_6621c6ec739a6.jpg">{{< figure src="blog_import_6621c6ec739a6.jpg" width="220px" height="165px" >}}</a>　<a href="blog_import_6621c6eca5cdf.jpg">{{< figure src="blog_import_6621c6eca5cdf.jpg" width="220px" height="165px" >}}</a></p><p><br/>３、オーダーは、セルフ方式</p><p> </p><p>　　ビールも、ジュースも、水も、完全にセルフ。</p><p> </p><p>　　冷蔵庫から勝手にとって飲み、会計時にボトルを数える。</p><p> </p><p>　　あるいは、自己申告する。</p><p> </p><p>　　この店、完全に客を信用しきっている。</p><p> </p><p>　　フードのオーダーも、会計も、キッチンカウンターで受け付ける。<br/>　<br/>　　ここでは、待っていても、何も始まらない。</p><p> </p><p>　　自分から動かなきゃだめだ。</p><p> </p><p><br/>４、圧倒的な回転率の低さ</p><p> </p><p>　　タケシは、この店に行くと５時間くらいいるらしい。</p><p> </p><p>　　他の客は、、？　というと、もっと長くいるらしい。</p><p> </p><p>　　奴の見立てでは、平均滞在時間５時間、あるいは６時間。</p><p> </p><p>　　ダラダラと閉店の１時までいる客が多い。</p><p> </p><p><a href="blog_import_6621c6ee5a1fc.jpg">{{< figure src="blog_import_6621c6ee5a1fc.jpg" width="220px" height="165px" >}}</a>　<a href="blog_import_6621c6f021019.jpg">{{< figure src="blog_import_6621c6f021019.jpg" width="220px" height="165px" >}}</a></p><p> </p><p>５、ハンパない数のバイク</p><p> </p><p>　　裏通りの店の前・横にずらっとバイクが駐められている。</p><p> </p><p>　　近隣にしてみれば迷惑だが、この店のオーナーは古くからの地主。</p><p> </p><p>　　そのうえ、バリの格闘技で腕を鳴らした猛者（親子）。</p><p> </p><p>　　誰も文句は言えないらしい。</p><p> </p><p><br/>６、夜中まで大音量のＢＧＭ</p><p> </p><p>　　バリ系ミュージックを中心にかけるＢＧＭは、夜中まで大音量。</p><p> </p><p>　　絶対、近隣に迷惑をかけているが、誰も文句を言わない。</p><p> </p><p>　　サヌールのバンジャールは煩い、って聞くけど、この店は放置されている。</p><p> </p><p><br/>７、圧倒的なローカル男子率</p><p> </p><p>　　この店の客の９５％はローカル男子。</p><p> </p><p>　　男同士で安酒を飲みながら、延々とだべる。</p><p> </p><p>　　タケシ曰く、「これがローカル男子の正しい遊び方」だとか。</p><p> </p><p>　　（ホントかよ）</p><p> </p><p>　　昨今、女性をターゲットにし、女性客が増えたらする、男もついてくる、</p><p> </p><p>　　そういう軟弱な発想のマーケティングをする店が多い中、この店は違う。</p><p> </p><p>　　そこを気に入っているとか。</p><p> </p><p>　　（僕は女の子が多い店がいいけどね）</p><p> </p><p>※もちろん、女人禁制、外国人禁制ではない。</p><p> </p><p><a href="blog_import_6621c6f1d3400.jpg">{{< figure src="blog_import_6621c6f1d3400.jpg" width="220px" height="165px" >}}</a>　<a href="blog_import_6621c6f362080.jpg">{{< figure src="blog_import_6621c6f362080.jpg" width="220px" height="165px" >}}</a></p><p> </p><p>８、ローカル若者男子、お行儀がいい</p><p> </p><p>　　安い酒場→若者→酒を飲んで騒ぐ→迷惑、というイメージになりがちだが</p><p> </p><p>　　ここはそうではない。</p><p> </p><p>　　若者のお行儀がとてもよい。</p><p> </p><p>　　ムリな飲み方はせず、ちびちびと楽しく飲んでいる。</p><p> </p><p>　　頼んだメニューがくるのが遅くても、文句を言う人もいない。</p><p> </p><p><br/>９、犬と猫</p><p> </p><p>　　店内は、犬と猫が数匹ウロウロしている。</p><p>　　<br/>　　これも行儀がいい。</p><p> </p><p>　　タケシは可愛いというが、奴は女性に好かれようと動物好きを演じている、</p><p> </p><p>　　と僕は見ている。</p><p> </p><p> </p><p>この店はサヌールで一番の人気酒場のようだ。</p><p> </p><p>あまりに多くのローカル若者男子がいることから、</p><p> </p><p>日本人がふらっと寄るには勇気が必要だ。</p><p> </p><p> </p><p>万一、僕が行くことがあっても、唐揚げは食べても、砂糖水アラックは飲まない。</p><p> </p><p>ホント、向こうの油ってカラダに悪そうなんだよな。</p><p> </p><p>ちなみに、１２月のツアーはまともな食事だからご安心を。。</p><p> </p><p> </p><p><a
