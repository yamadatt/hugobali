<p>今、１２月９日からのバリ島ツアーに向けて、スケジュールを考えている。</p><p> </p><p>どうせなら、皆さんに喜んでもらいたい。</p><p> </p><p>そこで、事前にアンケートを取り、要望や質問をした。</p><p> </p><p><br/>事前アンケートの内容は以下である。</p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝<br/> <br/>１、これまでバリ島へは何度、何日ほど滞在されて経験がありますか？<br/> <br/>２、バリ島ツアーの参加目的、期待することはどんなことですか？<br/> <br/>３、今回のツアーでは、どんな内容を盛り込んでほしいですか？<br/>　　※現在①～④を予定しております。<br/> <br/>　　①起業・投資などの話<br/> <br/>　　②豪華なヴィラ、建築現場の視察<br/> <br/>　　③旅行者の知らないディープなスピリチュアル体験<br/> <br/>　　④インドネシアの銀行口座・定期預金開設<br/> <br/> <br/>　　⑤その他の希望があればお書きください<br/> <br/>　　　（　　　　　　　　　　　　　　　　　　　　　　）<br/>　　<br/>　　⑥下記は希望者が多ければ組み入れ・組み換え可能です。<br/> <br/>　　　・日本とはスケールの違うラフティング<br/> <br/>　　　・アロママッサージ<br/> <br/>　　　・ウルワツ寺院のケチャダンス<br/> <br/>　　　・流行のビーチクラブ<br/> <br/>４、定期預金（年利６.５％くらい）をされたいですか？</p><p><br/>　　その場合、日本円でいくらほどですか？</p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p><br/>意外だったのは、銀行口座開設＆定期預金の要望が多かったことだ。</p><p> </p><p>インドネシアの銀行金利は、数年前と比べて下がったし、</p><p> </p><p>そもそも高金利が当たり前になるから、自分としては関心が薄い。</p><p> </p><p>しかし、一般的には魅力的に映るようだ。</p><p> </p><p> </p><p>よし、どうせなら、金利の高い銀行を探そう！！</p><p><br/>バリ島の中でどこの銀行の金利が高いのか？</p><p> </p><p> </p><p>インドネシアの銀行は、スポットで預金キャンペーンをやるし、</p><p> </p><p>オモテに出していない元金保証の運用商品があるから当日にならないと</p><p> </p><p>確実なことは言えない。</p><p> </p><p> </p><p>とりあえずは、大手銀行であるＢＣＡ、マンデリ、ＢＮＩを調べてみる。</p><p> </p><p> </p><p>まずは、高金利と言われるＢＣＡ。</p><p> </p><p><a
