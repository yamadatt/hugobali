<p>今日の株式セミナーの前に読者から寄せられた質問に長田氏が回答をくれた。</p><p> </p><p> </p><p><span style="font-weight: bold;"><span style="color: rgb(0, 0, 255);">寄せられた質問（Ｑ＆Ａ）</span></span></p><p> </p><p><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．誰にでもできる投資法ですか？</span></span></p><p> </p><p>Ａ．はい、特段のセンスがいらない投資方法です。</p><p> </p><p>    ただ、投資手法・自分の選択眼を信じて、数年単位で継続する力が必要です。</p><p> </p><p> </p><p><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．なぜ儲かる手法を公開するのですか？</span></span></p><p> </p><p>Ａ．それは三浦さんに頼まれたら、もう。笑</p><p> </p><p>　・・・ある程度手法がまとまってきたこともあり、頭の中を整理するために</p><p> </p><p>　丁度いい機会だと思いお受けしました。</p><p> </p><p>　なお、手法的には全く飽和しないのでライバルが増えるとかそういうことは気にしていません。</p><p> </p><p> </p><p><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．いくらくらいから始められますか？</span></span></p><p> </p><p>Ａ．正直多ければ多いほどいいのですが、個別株投資なので、１００万円はほしいです。</p><p> </p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．１００万円も貯められないのですが。</span></span></p><p> </p><p>Ａ．私が失敗から立ち上がり株式投資を開始した時に考えていたのは、</p><p> </p><p>　時間をかけて複利というパワーを使い、１，０００万、２，０００万というお金を作ることです。<br/>　<br/>　実際に私は、１０年かけて、まとまったお金を作ることに成功しました。<br/>　<br/>　逆にそれぐらいのお金を最終的に作るつもりがないのなら、</p><p> </p><p>　個人的には株式投資をやる意味を感じません。<br/>　<br/>　また、正直いって、最初のお金がないと雪だるまも膨らみません。</p><p> </p><p>　１００万円貯められる人は、また追加で１００万円貯められる人です。<br/>　<br/>　複利と追加資金投入で、まとまったお金がたまっていきます。</p><p> </p><p><br/>　もっとも、トレーディングが上手な人はもっと少ない金額で、</p><p>　</p><p>　かつ短期間でお金を効率よく増やすことができるのかもしれませんが、</p><p>　</p><p>　私にはその才能はありませんし（やって失敗しました）、ましてや教えることはできません。</p><p> </p><p>　爆発的におカネを増やしたい方は申し訳ないですが、ご期待に添うセミナーにはならないと思います。<br/>　</p><p> </p><p><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．投資で損するのが怖いのですが</span></span></p><p> </p><p>Ａ．そうですよね、その気持ちわかります。</p><p> </p><p>　　大損すると何も手につかなくなりますよ（笑）<br/>　　<br/>　　でも損する＝株価が変動するということなので、逆に上昇するというということでもあります。<br/>　<br/>　　また、損することに目が行きがちですが、投資していないことにより損している場合もあります。<br/>　　<br/>　　投資にタラレバは禁物ですが、リーマンショック後の２００９年、</p><p> </p><p>　　今から１０年前から日本株に投資していれば少なくとも２倍以上にはなっているはずです。</p><p> </p><p>　　さらに配当金も加わっているので実際には２倍以上でしょう。<br/>　　<br/>　　また、現物投資であれば、借金が残ることはありません。</p><p> </p><p>　　最大リスクが出資額に決まっているのです。</p><p> </p><p>　　（これが株式会社がここまで広まった理由です）</p><p> </p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．社会人になり立てです。投資経験がないのですが。セミナーの内容は理解できますか？</span></span></p><p> </p><p>Ａ．大丈夫です。</p><p> </p><p>　　わかりやすく具体例を使って一緒に勉強しましょう。<br/>　<br/>　　より深く知りたい方は、まずは何も考えず、早速インターネット専業証券会社に口座開いて、</p><p> </p><p>　　どんな株でもいいからすぐ１単元買いましょう。</p><p> </p><p>　　５万〜１０万もあれば十分です。<br/>　　<br/>　　本を何十冊読んでも理解できないことが急に頭に入ってきます。</p><p> </p><p>　　経済ニュースや社会の動きにも敏感になります。　</p><p> </p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．親や親せきに株式投資は危ないから手を出してはいけないといわれました</span></span></p><p> </p><p>Ａ．かつては証券会社もすこしやんちゃなところがあり、</p><p> </p><p>　　お客さんの資産を大切に扱っていなかった時代もありました。<br/>　<br/>　　証券会社に口座をもっていると、変な株や金融商品を進められたり、</p><p> </p><p>　　信用取引で大損したという話をどこかでお聞きになった、</p><p> </p><p>　　またはご自身で経験されたのかもしれません。<br/>　<br/>　　また、日本はバブルが発生して過度に株価が上昇してしまったこともあり、</p><p> </p><p>　　長らく株価が低迷することとなりました。<br/>　</p><p><br/>　　しかし、今はインターネット専業証券であれば営業は全くされません。</p><p> </p><p>　　また、日本含め株価はここ１０年で世界的に上昇基調です。</p><p> </p><p>　　上昇しているのですから、今も長期投資している人は少なくとも大損はしていないはずです。<br/>　<br/>　　株式投資が危ないのではなくて、投資のやり方が危なかったのです。</p><p> </p><p>　　インターネット専業証券なら、投資タイミングをすべて自分で選択することもできるし、</p><p> </p><p>　　定期的に積み立てていく方法もありますので、リスクを管理しながら投資することが可能です。</p><p> </p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．英国のＥＵ離脱とか米中戦争とかで世界経済の行き先が不安です</span></span></p><p> </p><p>Ａ．そんな評論は専門家（金持ちではない）に任せておきましょう。</p><p> </p><p>　　我々庶民は何も打つ手立てはありません。</p><p> </p><p>　　毎日電車に乗って、上司にお小言いわれながら、いうこと聞かない部下の面倒みて、</p><p> </p><p>　　ちょっとした手直しで資料作り直しに２時間残業して、必死に給料稼ぐしかないんです。</p><p> </p><p>　　家でも家事に育児に忙しいんです。</p><p> </p><p>　　トランプや習近平よりもっとご機嫌を取らないといけない相手がいるんです。</p><p> </p><p><br/>　　世界経済なんて大それたことは語れませんが、これまでも戦争だの、</p><p>　</p><p>　　オイルショックだの、テロだのもっとヤバい局面はありました。<br/>　<br/>　　経済もナンタラショックに何度見舞われたことかわかりません。<br/>　<br/>　　でも、世界で最も参考にされている指標の一つ、米国ダウ３０種工業平均は、</p><p>　</p><p>　　１９５０年１月末２００ドルだったものが２０１９年１月末現在で２５，０００ドルです。<br/>　</p><p><br/>　　確かに、一時的に米中戦争の長期化により経済活動に滞りが生じ、</p><p>　</p><p>　　結果リセッションになり株価が暴落するかもしれません。<br/>　<br/>　　仮に起きた場合、どれぐらい暴落するかもわかりません。</p><p>　</p><p>　　いつまで暴落するかもわかりません。<br/>　<br/>　　しかし、世界中が資本主義経済で回っている以上、将来的には経済の膨張がつづき、</p><p>　</p><p>　　株価は上昇していきます。<br/>　<br/>　　私は資本主義の発展を信じて長期投資していますが、とにかく損失が怖い方は</p><p>　</p><p>　　無理に投資する必要はないと思います。</p><p>　</p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．４０代、５０代でもできる投資手法ですか？</span></span></p><p> </p><p>Ａ．もちろん大丈夫です。</p><p>　</p><p>　　ただ、２０代、３０代と比べると、様々な出費もかさみ、</p><p>　　</p><p>　　将来への備えのためにリスクを抑えていかなければならない一方、<br/>　<br/>　　投資にかけられる生涯時間も減っていきますので、</p><p>　</p><p>　　一般的には、慎重に資産配分されたほうがいいです。</p><p>　</p><p>　　相場の上昇時期はコントロールできないので、</p><p>　</p><p>　　年金が主たる収入源の年代の方にはあまりオススメしない投資法です。</p><p> </p><p><br/><span style="font-weight: bold;"><span style="color: rgb(255, 0, 0);">Ｑ．株で稼いでいる人って聞かないのですが、本当に儲かるんですか？</span></span></p><p> </p><p>Ａ．お金を持っている人ほど、株式や不動産で儲けてます。</p><p> </p><p>　だって、こんなに株価が上がっているんですよ。</p><p>　</p><p>　あなたの周りにも株式投資で儲けている人、たくさんいますよ。</p><p>　</p><p>　日本はお金に関しては陰湿な妬み社会なので、</p><p>　</p><p>　周囲からねたまれるのがめんどくさい、怖いから言わないだけです。</p><p>　</p><p>　自慢していいこと何もないですから。</p><p> </p><p> </p><p> </p><p> </p><p><span
