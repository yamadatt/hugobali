style="font-size: 1em;"><span style="color: rgb(0, 0, 255);">長田氏の株式投資実践会「個別フォローLineグループ質問集」その１</span></span></p><p> </p><p><span style="color: rgb(0, 0, 0);">質問１</span><br/>（「会社の事業内容を調べる時は、説明会用資料を確認して下さい」とのことですが）<br/>会社説明会用資料はどうやって見つけるのですか？</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問４</span><br/>会社のＩＲに「下期は成長投資をするため業績予想は低くなる」とあったのですが、<br/>これは社長の「自信を持って成長投資する！」の言葉通り受け取ってのでしょうか？<br/>それでも、業績悪いとやっぱり株価は下がるものですか？</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問５</span><br/>下がっているZOZOTOWN（3092）ですが、もう買ってもいい水準ですか？<br/>見通しを教えてください。</p><p> </p><p> </p><p><a href="blog_import_6621cc881c887.png">{{< figure src="blog_import_6621cc881c887.png" width="776px" height="779px" >}}</a></p><p> </p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問１【Ｍ氏】<br/>（「会社の事業内容を調べる時は、説明会用資料を確認して下さい」とのことですが、）<br/>会社説明会用資料はどうやって見つけるのですか？</span></p><p> </p><p>【長田先生】</p><p>必ず会社のウェブサイトにir資料をまとめたコーナーがあります。<br/>そこから、決算説明会資料だったり、決算補足説明資料という名前の資料を探してみてください。<br/>この資料は法律で作成が義務づけられているわけではないのですが、分かりやすく決算の内容を伝えるため、<br/>ほとんどの上場企業が決算毎、少なくとも半期に一度は更新しています。</p><p><br/>また会社のウェブサイトには個人投資家向けの会社案内ページがありますので、<br/>それも理解を深めるための資料になります。</p><p><br/>【Ｍ氏】</p><p>ありがとうございます。<br/>そこまで調べて投資するのですね。<br/>もはや、、仕事ですね。<br/>ガチで稼ぐというのはそういうことなのですね</p><p><br/>【長田先生】</p><p>そうですね、そうしないと再現性もないですし笑<br/>会社の業務内容を調べているとどうしても財務の話に触れざるを得ないのですが、<br/>そこらへんも今後フォローしていきますね。</p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問２【Ｎ氏】</span></p><p><span style="color: rgb(0, 0, 0);">どこから手をつけて良いのかまだまだ手探り状態ですが、<br/>少しずつ資料なども見られるようになりたいです</span></p><p><br/>【長田先生】</p><p>まずはセミナーでやった通り増収増益企業を探しましょう。出来れば配当金を一定に出している会社がいいです。<br/>必ずしも時価総額やオーナー企業にこだわる必要はありません。</p><p>あとでまたいくつか候補銘柄お伝えしますね</p><p><br/>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問３【Ｎ氏】</span></p><p><span style="color: rgb(0, 0, 0);">今さらですみません教えてください<br/>増収は売上高が増えているかどうかだと理解しています<br/>増益は何利益を見ていけばよいでしょうか<br/>一株利益？経常利益？</span></p><p><br/>【長田先生】</p><p>営業利益をみてください。営業利益は本業の儲けを示す数字なので、本業の成長性がわかります。<br/>経常利益は配当金や家賃収入などを加えた数字です。<br/>一株利益は税金を引いた後の純利益を発行済み株式数で割ったものです。</p><p>当期の純利益が1億円で、発行済株式数が1千万円なら一株あたりの純利益は10円ですか</p><p>セミナーは税引き後の利益を見るようお話ししたような気もしますが、営業利益の方がより適切でしたね。</p><p><br/>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><span style="color: rgb(0, 0, 0);">質問４【Ｎ氏】</span></p><p><span style="color: rgb(0, 0, 0);">私も先ほど7320のIRを見てみました<br/>難しくて半分くらいしか理解できなかったですが(emoji)<br/>その中に、下期は成長投資をするため業績予想は低くなると書かれてあったのですが、<br/>これは社長の、自信を持って成長投資する！の言葉通り受け取ってよい内容なのでしょうか<br/>予想通りだとしても、業績悪いとやっぱり株価は下がるものですか？</span></p><p><br/>【長田先生】</p><p>ほとんどの個人投資家は表面的な最終利益しかみていないので、大体下がります。<br/>成長が期待されている銘柄であればなおさらです。</p><p> </p><p>【Ｎ氏】<br/>なるほど<br/>それなら下がったら買いやすくなりますね(^｡^)</p><p> </p><p>【長田先生】<br/>繰り返しになりますが、この会社は見た目以上に儲かっています。毎年ストックとしての収益が積み重なっているのです。<br/>もちろん50億円もない小型株なので、価格は動きます。<br/>短期的な業績悪化懸念で売られたらチャンスです。もちろん分割売買でさらに売られた場合に備えます。<br/>この銘柄なら900円700円500円ぐらいの心持ちでいればいいかと思います。</p><p>あと伸びる会社の特徴は本業のノウハウをうまく使って新規事業としてシステム開発したり、<br/>コンサルタント業務したり、アウトソーシングしたり人材派遣はじめたりします。</p><p>ちょっと違う業態ですが、イーギャラティ8771という会社も手堅い商売やっています。<br/>オーナー企業ではないし、上場10年以上たっているし、時価総額も300億円越えていますが、増収増益を続けています。<br/>この会社も随分上がってしまい、その分大化けの期待性はなくなってますが。<br/>また明日ご紹介します。</p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><a style="color: rgb(0, 0, 0);">ZOZOTOWN（3092）は、すでに買い下がる2000を切っているので<br/>もう買っちゃってもいいんじゃないかという水準なんでしょうか？</span></p><p><br/>【長田先生】<br/>先日ご紹介した日本リビング保証よりも予想がつきやすいですが、その分上昇余地は限られます。<br/>見通しが明るいと株価は上がってしまいますので。<br/>ただこれも保証業務という割合地味な業務内容なのでこの水準で収まっています</p><p><br/>いつかっても大丈夫ですが、一年ぐらい上昇しないことを覚悟してください。<br/>すべては前沢社長の構造改革次第なのですが、あまりにもゴタゴタしてしまったので取引先との関係など数ヶ月で改善できるとは思えません。</p><p>プラットフォームとしてはたいへん優秀です。後発でここまで成長してきたので。<br/>今は逆風が吹いていますが、再び成長路線に転じるとコロッと雰囲気が変わるのが株式相場です。<br/>ただ、時価総額がすでに相当大きい（6000億円)ので5倍10倍になることは期待しにくいです。5倍で時価総額3兆円、10倍で6兆円です！</p><p><br/>【Ｔ氏】<br/>ありがとうございます。<br/>あまりに巨大だと数倍になる夢が見づらくなっちゃうんですね！</p><p><br/>【長田先生】<br/>そうですね、すでに4000円は達成しているので、これくらいの株だと良くても5000円くらいでしょうか。当面は。</p><p>とにかく逆風がやむまで応援するにつきます。この銘柄の場合。</p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p><a
