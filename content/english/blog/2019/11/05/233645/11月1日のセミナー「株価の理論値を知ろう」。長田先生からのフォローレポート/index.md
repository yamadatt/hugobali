<p>昨日の記事でも書いたが、長田先生から参加者へのフォローレポートが届いている。</p><p> </p><p>昨日の記事</p><p><a href="entry-12542344229.html" target="_blank">長田先生、あなたはすごいです。普通はここまでやれないよ。</a></p><p> </p><p> </p><p>かなりの分量（Ａ４で１０枚超）のレポートだ。</p><p> </p><p>実は、明日からバリ島に行くのだが、早朝に成田空港に行かなくてはならない。</p><p> </p><p>しかし、なにも用意をしていない。</p><p> </p><p>それどころか、まだ会社にいて残った仕事を片付けている。</p><p> </p><p><br/>というわけで、長田先生のレポートの推敲は諦めた。</p><p> </p><p>とりあえず１つだけは、そのままアップする。</p><p><br/>あとは、参加者の方へはそのまま送る。</p><p> </p><p>ブログの方は、これでご勘弁を。。</p><p> </p><p> </p><p>さて、時々「このブログは、出し惜しみしないですよね」と言われることがあるが</p><p> </p><p>そんなこと、考えたこともない。</p><p> </p><p>このレポートにしても、晒すことに長田先生も僕も躊躇はない。</p><p> </p><p>レポート以上のことをセミナーでは提供することができるし、</p><p> </p><p>僕らは成長し続けるから、現時点のレポートなど出したところで</p><p> </p><p>それを超えていくのは間違いないからだ。</p><p> </p><p> </p><p>時間もないので、能書きはそれくらいにしてレポートを紹介する。</p><p> </p><p>ちなみにこれで、全体の１／４くらいだ。</p><p> </p><p> </p><p>＝＝＝＝＝　「株価の理論値を知ろう」フォローレポート　＝＝＝＝＝＝</p><p> </p><p>こんにちは、長田です。</p><p> </p><p>先日のセミナーの復習メモを送付します。当日は時間の関係もあり、</p><p> </p><p>話せなかった内容についても少し補記しております。ご参考になれば幸いです。</p><p> </p><p>少し量が増えてしまいましたので、複数回に分けて送付いたします。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">株価は需給</span></p><p> </p><p>・株価は究極的には需給で決まります。</p><p> </p><p>  中村超硬（6166）は、業績悪化懸念から画期的技術（補足）を開発したことが</p><p> </p><p>  カタリスト（株価を大きく動かすきっかけとなるもの）となり、株価が乱高下しています。</p><p> </p><p>  霞が関キャピタル（3498）も大きく株価が動いています。</p><p> </p><p> </p><p>・その間に企業の価値に大きな違いが生じたでしょうか？</p><p> </p><p>  将来の業績に対する期待感は大きく増えたり減ったりしたかもしれませんが、</p><p> </p><p>  現在の会社の価値がいきなり大きくなったわけではありません。</p><p> </p><p> </p><p>・買いたい人が多ければあがりますし、売る人が多ければ下がります。</p><p> </p><p>  なお、買いが多いとか売りが多いとは、正確にいえばどちらの勢いが</p><p> </p><p>  強いかということです。</p><p> </p><p>  買いと売りは常に１対１で取引されるのだから、数量自体は同じです。</p><p> </p><p> </p><p>・さらにいえば、価格が動くというのは、どちらのほうが取引を急いでいるか</p><p> </p><p>　ということでもあります。</p><p> </p><p>　急いでいる人は、高い値段で買うし、安い値段で売ることになるのです。</p><p> </p><p> </p><p>・お茶を買うシーンを思いだしてみましょう。</p><p> </p><p>　コンビニってディスカウントスーパーよりも高いですよね。</p><p> </p><p>　時間があればドンキホーテとか安いところでまとめて買います。</p><p> </p><p>　急いでいるということは、高くつくのです。</p><p> </p><p> </p><p>・急いでいる人から、急いでいない人へと富が移転していくというのは</p><p> </p><p>　株式市場の大原則ですので、覚えていてください。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">株価は振り子時計</span></p><p> </p><p>・話が脱線しましたが、究極は需給できまるのが株価なのですが、</p><p> </p><p>　実は株価はあるべき価格（一般的には理論価格と呼ばれています）を中心に</p><p> </p><p>　上下に動いているのです。</p><p> </p><p>　それはあたかも振り子時計の振り子が右に左に動き続けるように。</p><p> </p><p>　あるべき価格を通過することはありますが、決してその価格にとどまることは</p><p> </p><p>　ありません。</p><p> </p><p> </p><p>・そして振り子の振れ幅は、銘柄によって違いますし、その時々の市場動向によっても</p><p> </p><p>　異なりますので、正確に振れ幅を知ることはできません。</p><p> </p><p>　すなわち、株価の底や天井を充てることはできません。</p><p> </p><p><br/><span style="color: rgb(255, 0, 0);">様々な理由で揺れ動く株価</span></p><p> </p><p>・成長株であれば、毎年利益は基本的に増えていきますから、</p><p> </p><p>　株価は理屈の上では右肩上がりになるはずです。</p><p> </p><p>　ただし、その通り株価は動くほど単純なものではありません。</p><p> </p><p>　様々な理由で上下に動きます。</p><p> </p><p>　以下その数例をご紹介しましょう。</p><p> </p><p> </p><p>・外国為替の動向は株価、特に輸出企業の動向に大きな影響を与えます。</p><p> </p><p>　輸出企業はドルで取引の代金をもらいますから、それを日本円に戻すときの</p><p> </p><p>　ドル円レートが円安ドル高のほうが、業績が良くなるのです。</p><p> </p><p>　一方内需系（NTTドコモ、セコム、東京ガスなど）の銘柄は</p><p> </p><p>　為替動向で業績が左右されにくいです。<br/>　</p><p>　といっても全体の相場は基本的にはTOPIX、日経平均株価などの指数によって</p><p> </p><p>　左右される局面が多いです。</p><p> </p><p>　指数というものはどのようにつくっても時価総額が大きい銘柄の影響が大きくなります。</p><p> </p><p>　時価総額が大きい銘柄は輸出企業が多いので、結局は輸出系企業の動向である程度</p><p> </p><p>　内需系の会社の株価も動いてしまうということになります。</p><p> </p><p> </p><p>・外国人投資家の売買動向も株価に大きな影響を与えています。</p><p> </p><p>　というのも、現在証券取引所で取引されている売買高の6割～7割を</p><p> </p><p>　外国人投資家が占めているからです。</p><p> </p><p>　多数派の動向が全体の相場に与える影響は当然大きくなります。</p><p> </p><p> </p><p>・なお、外国人投資家というのは、外国人の機関投資家のことです。</p><p> </p><p>　習慣的に外国の機関投資家のことを外国人投資家と呼ぶ習慣があります。</p><p> </p><p>　また、この外国人というのは実は海外を拠点とする日本の機関投資家のお金も入っています。</p><p> </p><p> </p><p>・税制上の観点からある程度お金持ちになったら、税制上の優遇を受けるため、</p><p> </p><p>　海外でファンドを立ち上げる人もいます。</p><p> </p><p>　そういう人たちは実質的には個人投資家ですが、統計上は外国人投資家に入ります。</p><p> </p><p> </p><p>・自然災害は先般の台風や、東日本大震災などの災害が発生した場合です。</p><p> </p><p>　こうした天災が発生すると、短期的に問答無用で株価は暴落します。</p><p> </p><p>　これからも日本は天災と付き合っていかなければなりませんので、</p><p> </p><p>　定期的に自然災害を理由とした株価の暴落は発生するでしょう。</p><p> </p><p> </p><p>・金利も企業業績に影響を与えます。金利があがると、企業はお金が借りにくくなるし、</p><p> </p><p>　金利負担が増えるので企業業績にマイナス。</p><p> </p><p>　逆に金利が下がると、企業はお金が借りやすくなるし、金利負担が減るので企業業績にマイナス。</p><p> </p><p> </p><p>・住宅ローンを考えてみればわかりやすいと思います。</p><p> </p><p>　金利が下がると、お金を借りやすくて、家を買いやすくなる。</p><p> </p><p>　一方で、金利が上がると、お金を借りにくくなり、家が買いにくくなる。</p><p> </p><p> </p><p>・なお、増税も金利と同様に費用の増加につながりますので、企業業績を圧迫します。</p><p> </p><p>・債券の価格は金利が上がると債券価格自体は下がります。</p><p> </p><p>　逆に金利が下がると、債券価格自体は上昇します。</p><p> </p><p>　これ、最初はわかりにくいので、今わからなくても気にしないでください。</p><p> </p><p>　何回でも説明しますので。</p><p> </p><p> </p><p>・利回りが上昇する＝その商品の価値が下がっているというのは、</p><p> </p><p>　株でも、不動産でも、債券でも皆同じです。</p><p> </p><p> </p><p>・国内の政治状況も株価に大きな影響を与えます。直近ではアベノミクス、</p><p> </p><p>　史上最大級の金融緩和が続いている状況ですので、リスク資産にお金が入ってきます。</p><p> </p><p>　安倍政権が誕生する前の民主党政権では、投資家にとって逆風となる政策ばかりが</p><p> </p><p>　導入されておりましたので、株価は上がりませんでした。</p><p> </p><p>　安倍首相率いる自民党が政権を取ったことにより、投資家が安心して投資できるよう</p><p> </p><p>　環境が変わったのです。</p><p> </p><p> </p><p>・日本銀行による、大量の資金供給による円安誘導、金利引き下げ（マイナス金利の導入）、</p><p> </p><p>　債券の買い入れのみならず、J-REIT（不動産投資信託。比較的少額から不動産投資を</p><p> </p><p>　するのにうってつけの商品です。）、ETF（投資信託を取引中に購入できるようにしたもの。</p><p> </p><p>　日経平均やTOPIXのほか、様々な指数に連動する商品が上場しています。）を</p><p> </p><p>　通じたリスク資産の買い上げ。</p><p> </p><p>　さらに国内の年金ファンドが大量に株式を買い上げたので、株価が上昇したのです。</p><p> </p><p>　また、ここら辺の基本的な用語説明は次回以降どこかでやります。</p><p> </p><p> </p><p>・もちろん景気動向も株価動向に影響を与えます。</p><p> </p><p>　一番手堅い方法は、株なんてだれも見向きもしないようなとき、景気が悪い時に株を買うことです。</p><p> </p><p>　これは数年単位での勝負になりますので、長期間で株式投資を考えている方は参考にしてください。</p><p> </p><p> </p><p>・そして真ん中にある企業業績。これが株価に一番大きな影響を与えます。</p><p> </p><p>　最終的には儲かっている会社の株は高くなるし、もうけが少ない会社の株は安くなる。</p><p> </p><p> </p><p>・もちろん株価は企業業績以外の要因（外的要因）の影響を受けて、全体相場が下がるときは</p><p> </p><p>　どんな優良企業の株でも下がります。</p><p> </p><p>　ただ、下がり幅が少ないのです。</p><p> </p><p>　そして上がるときには他の株よりもよく上昇するのです。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">株式の投資利回りは高い</span></p><p> </p><p>・株式投資の利回りは高い。</p><p> </p><p>　歴史的に見て株式投資の利回りは年間約5％～9%の間で推移してきました</p><p> </p><p>　（ここでいう利回りは平均株価をさします）</p><p> </p><p> </p><p>・つまり資産運用の手段としては、株式投資が一番有利なのです。</p><p> </p><p>　不動産投資よりも有利な投資なのです。</p><p> </p><p>　もちろん、利回りが大きいということの裏側にはそれなりの理由があります。</p><p> </p><p>　それは含み損や含み益に耐えながら長期的に保有金額を増やしていかなければ</p><p> </p><p>　ならないということです。</p><p> </p><p> </p><p>・不動産は大体の取引価格はありますが、相対取引ですので、購入している人が</p><p> </p><p>　時価を意識することはありませんし、取引が株式投資ほど活発に行われておりませんので、</p><p> </p><p>　価格の推移がゆっくりです。</p><p> </p><p>　しかしながら、株式は毎日取引所で取引しようと思えばいつでも取引ができる。</p><p> </p><p>　取引の流動性が非常に高いのです。だから、買いと売りのバランスが崩れると、</p><p> </p><p>　流動性があるゆえに価格が大きくぶれる。</p><p> </p><p> </p><p>・収益の上がり方という面からも違いがあります。不動産が生み出す収益とは、</p><p> </p><p>　賃料ですので、安定しています。</p><p> </p><p>　家を借りるときの賃料って借りている間はずっと同じですよね。</p><p> </p><p>　だから、賃料を利回りで割って導き出される不動産価格はある程度安定しているのです。</p><p> </p><p>　借金や相続で首が回らなくならない限り持ち続けていれば、ゆっくりと元本を回収できますので、</p><p> </p><p>　売る方もそんなに安い値段で売る理由がありません。</p><p> </p><p>　したがって、株式のように急激には下がりません。</p><p> </p><p> </p><p>・一方、株式投資は会社の業績によって大きく影響を受けます。</p><p> </p><p>　今年儲かった金額と来年儲ける金額が同じくらいであるという確率は不動産に比べると低い。</p><p> </p><p> </p><p>・したがって、株の利回りが5%～9%といっても、その利益が毎年大きくぶれるのですから、</p><p> </p><p>　不動産でいうところの物件価格＝株価も大きく動くのです。</p><p> </p><p> </p><p>・株式投資で資産を増やすとは、短期的な株価変動を乗り越える我慢料です。</p><p> </p><p>　他の投資商品とは比べ物にならないぐらいの価格変動があることを覚悟してください。</p><p> </p><p>　株価がマイナスになるのは、本当によくあることですので、</p><p> </p><p>　短期的な株価の動向に一喜一憂していたらやっていられません。</p><p> </p><p> </p><p>＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝</p><p> </p><p> </p><p> </p><p> </p><p><span
