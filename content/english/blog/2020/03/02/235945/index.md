<p>初歩的なことかもしれないが、株の売買注文の方法について質問されることがある。</p><p> </p><p>「成行注文」と「指値注文」。</p><p> </p><p>どちらがいい、ということはないが、その考え方について長田先生からメッセージが届いた。</p><p> </p><p><br/>＝＝＝＝＝　長田先生からのメッセージ　＝＝＝＝＝＝＝＝＝＝</p><p><br/>【売買注文方法をどうやって使い分けるか】</p><p> </p><p>情報処理技術が発展したことにより、今や〝逆指値″というリスク管理に</p><p> </p><p>優れた売買注文方法など、個人投資家でも様々な注文形態で発注できる</p><p> </p><p>ようになりました。</p><p> </p><p>便利になった反面、下記が分からないという声をいただきます。</p><p> </p><p>・どの注文方法を選択すればいいのか</p><p> </p><p>・どのように使い分けたらいいのか</p><p> </p><p> </p><p>そこで、今回は中・長期投資家の視点で売買注文方法について解説します。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○株式の注文について</span></p><p> </p><p>株式の注文には、大きく分けて成行（なりゆき）注文と、指値（さしね）注文があります。</p><p> </p><p>その他の注文方法は、中・長期の投資家は使うことがないので、考えなくて構いません。</p><p> </p><p>私は買い・売りどちらも大体寄付成行注文を出しています。</p><p> </p><p>午前9:00前か、午後12:30前で相場が開いていない時間帯に成行注文を出すと、</p><p> </p><p>寄付の成行注文として取引所に発注されます。</p><p> </p><p>取引所が開いている時間帯に取引する場合には、価格が飛ぶのが怖いので、指値を使います。</p><p> </p><p>買いを出すときは前日が安く終わった日、売りを出すときは前日が高く終わった日に売買します。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○成行注文とは</span></p><p> </p><p>成行注文とは、市場に出ている注文のうち一番安い注文に対して売買する注文方法です。</p><p> </p><p>100円で「買い」、101円で「売り」気配が出ている場合、成行で買い注文を出すと、</p><p> </p><p>101円で売買が成立します。</p><p> </p><p>自分の発注数量が多いと、流動性が少ない場合には株価を動かしてしまうこともありますが、</p><p> </p><p>分割売買する中・長期の投資家は気にすることはありません。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○成行注文のメリット</span></p><p> </p><p>成行注文のメリットは、市場に注文が出ていれば必ず約定するということです。</p><p> </p><p>指値をしていてもそこまで価格が下がらずにどんどん株価が上がっていってしまい、</p><p> </p><p>悔しい経験をしたことがある方は少なくないと思います。</p><p> </p><p>成行で注文すれば必ず約定するのですから、「買い逃がす」または「売り逃す」</p><p> </p><p>ということはありません。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○成行注文のデメリット</span></p><p> </p><p>成行注文のデメリットは、売買注文の数量が少ないと不利な価格で約定する</p><p> </p><p>可能性があるということです。</p><p> </p><p>例えば、100円買い、105円売りの気配で買い注文を出すと、105円で約定します。</p><p> </p><p>普段だったら大体101円で約定するところ、たまたま売買数量が少ないときに発注すると、</p><p> </p><p>価格が飛んでしまうのです。</p><p> </p><p>こういった現象は売買高が少ない新興株によく見られます。</p><p> </p><p> </p><p>また現在の東証の売買システム（arrowhead）は1000分の1秒単位で売買注文を</p><p> </p><p>さばいているので、実際に見てからでは遅い場面もあります。</p><p> </p><p>成行注文を出したとしても、買おうと思っている注文が残っているかはわからないのです。</p><p> </p><p>あなたが注文を出して、その注文が証券取引所のシステムに送信されるそのわずかな時間で、</p><p> </p><p>誰かがあなたの買おうと思っていた株を横からかっさらっていってしまうかもしれません。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○指値注文とは</span></p><p> </p><p>指値注文は、自分で売買する価格を決めて発注する方法です。</p><p> </p><p>売買する値段が決まっているので、予想外の価格で約定することはありません。</p><p> </p><p>100円で買うという注文を出しておけば、約定するのは必ず100円です。</p><p> </p><p>101円とか、105円で約定することはありません。</p><p> </p><p>場合によっては、99円、98円など100円より安い値段で約定することがあります。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○指値注文のメリット</span></p><p> </p><p>指値注文はその日だけでなく、先日付でもずっと注文を残しておくことができるので、</p><p> </p><p>２週間後まで有効な注文として発注することができます。</p><p> </p><p>例えば株価が今100円だとして、80円で指値注文を出しておくと、発注する証券会社は</p><p> </p><p>２週間ずっと80円で注文を出し続けてくれるのです。</p><p> </p><p>（正確には毎日注文が失効しますので、毎日新しく注文を80円で出してくれます）</p><p> </p><p> </p><p>株式相場を見ている必要もないので、忙しい方でも週末に一度注文をだしておけば、</p><p> </p><p>あとは約定まで株式市場を見ておく必要はありません。</p><p> </p><p>また、冷静な時に指値注文を出しておけば、価格が下がってきたとしても</p><p> </p><p>冷静に売買することができます。</p><p> </p><p>株価がいよいよ下がってきて、買いたい価格に近づいてきたとしても、</p><p> </p><p>そういう展開で株式の売買発注を出すのは怖くなるものなのです。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○指値注文のデメリット</span></p><p> </p><p>指値注文のデメリットは注文が約定しない可能性があるということです。</p><p> </p><p>ですから、どうしても株がほしいという状況においては、自分の思った通りに</p><p> </p><p>売買が成立しない可能性が残ります。</p><p> </p><p>デメリットというほどでもないのですが、一度指値を決めたら、その指値は</p><p> </p><p>動かさないほうがいいです。</p><p> </p><p>最初の指値は冷静であり、修正した指値は思惑が入った（欲が入った）指値だからです。</p><p> </p><p>もう少し安くなるだろう、もう少し高くなるだろう、という考えで指値を動かしていると、</p><p> </p><p>いつも冷静に取引することができなくなります。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○長田はなぜ寄付成行注文を使うのか？</span></p><p> </p><p>上記のような特徴がある成行・指値注文のうち、私は寄付成行注文を使います。</p><p> </p><p>寄付注文の特徴は、値段が一本値で決まること、そして沢山の売り買いが交錯するので、</p><p> </p><p>大量の注文も比較的スムーズに発注できるからです。</p><p> </p><p>値段が一本値で決まるというのは、スプレッドを支払わなくてもいいということです。</p><p> </p><p>スプレッドとは何でしょうか？</p><p> </p><p>ザラバ（日中の売買）取引では、買いと売りの気配値に価格差があります。</p><p> </p><p>100円以下で買いたいズラっと注文を出し、101円以上で売りたい人がズラっと注文を</p><p> </p><p>出しています。</p><p> </p><p> </p><p><a href="blog_import_6621d792d8ecf.png">{{< figure src="blog_import_6621d792d8ecf.png" width="220px" height="219px" >}}</a></p><p> </p><p> </p><p>その様子を表したのが板（いた）です。</p><p> </p><p>昔は相場の価格変動を実際に板に書いていたので、その名残で板というのです。</p><p> </p><p> </p><p>話が少しそれましたが、注文を約定させたいほうが必ずスプレッドを支払わなければ</p><p> </p><p>ならないのです。</p><p> </p><p>例えば、101円ウリ、100円カイという気配があったとすると、買いたい人は１０１円の</p><p> </p><p>注文を取りに行って１円高く買わなければなりませんし、売りたい人は１００円で</p><p> </p><p>１円安く売らなければなりません。</p><p> </p><p>ところが、寄付（と大引）の売買注文では板寄せ売買といって、買い注文と売り注文の</p><p> </p><p>数量をそれぞれまとめて、数量が釣り合うところで価格を決める方法を採用しています。</p><p> </p><p>だから、スプレッドを支払わなくてもいいのです。</p><p> </p><p> </p><p>詳しい値段の決め方は細かくなってしまうのでここでは割愛しますが、</p><p> </p><p>１万株の買い注文と、１万株の売り注文を同じ値段、100円なら100円でマッチングさせる取引です。</p><p> </p><p>特に、朝の寄付は多数の投資家からの売り買いの注文がまとめて取引所に発注されていますから</p><p> </p><p>多めの株数で成行注文をだしても、著しく不利な値段で約定することが少ないのです。</p><p> </p><p>といっても、なるべく買うときは安く買いたいし、売るときは高く売りたいので、</p><p> </p><p>前日の動向を加味して注文を出しています。</p><p> </p><p> </p><p>また、このように取引をすることで、精神的に考える要素が一つ減ります。</p><p> </p><p>「売買注文は寄付成行で出す」と決めておけば、迷いが一つ減ります。</p><p> </p><p>短期投資では少しの価格差が大差になりえますが、中・長期投資になればなるほど</p><p> </p><p>１円や２円の違いはあまり気にする必要はないからです。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○分割売買と株式注文方法</span></p><p> </p><p>分割売買をする時にキッチリと120円100円80円と注文してもいいですが、</p><p> </p><p>時間軸をしっかりとっておけば成行注文でも全く問題ありません。</p><p> </p><p>ただ、成行と指値をどう使うかはスタイルの違いですから、どちらでも精神的に楽な方法を</p><p> </p><p>選んでください。</p><p> </p><p>それよりも、買いも売りも、なるべく分割するということを忘れないようにしてください。</p><p> </p><p> </p><p><span style="color: rgb(255, 0, 0);">○まとめ</span></p><p> </p><p>取引手法には大きく分けて、成行と指値という方法があります。</p><p> </p><p>どちらにもメリット・デメリットがあるのでそれぞれの特徴を理解して発注してください。</p><p> </p><p>朝の寄付は沢山の注文が交錯するため流動性が高いことと、板寄せ形式で価格が決まるので</p><p> </p><p>スプレッドを支払う必要がないという２つの特徴があります。</p><p> </p><p>私は基本的には寄付成行で注文を出しています。</p><p> </p><p>もちろん指値で注文することもあります。</p><p> </p><p>なお、寄付・大引・玉・踏み上げといった独特の株式相場用語は、江戸時代の遊郭・吉原の</p><p> </p><p>用語にその語源があります。</p><p> </p><p>これらのお話はまた懇親会で・・・</p><p> </p><p> </p><p> </p><p><span
